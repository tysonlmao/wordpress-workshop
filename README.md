# wordpress-workshop
This is a testing environment for using the Wordpress environment.

<img src="linus.png"
     alt="linus"
     style="float: center; margin-right: 10px;" />

## Usage
```bash
# start and stop the wordpress environment
wp-env start 
wp-env stop
```

```bash
# start and stop the live block environment (reloads on save)
npm run start
npm run stop
```
